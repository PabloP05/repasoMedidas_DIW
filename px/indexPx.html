Esta pagina esta diseñada usando unidades de medida en px (pixeles). Un pixel es la unidad de medida mas pequeña que se puede utilizar en una pantalla. El uso de pixeles permite un control preciso sobre el tamaño y la posicion de los elementos en la pagina web. Sin embargo, el uso exclusivo de pixeles puede afectar la adaptabilidad y accesibilidad de la pagina en diferentes dispositivos y resoluciones de pantalla. Es importante considerar el uso de unidades relativas como porcentajes o ems para mejorar la experiencia del usuario en una variedad de dispositivos.
